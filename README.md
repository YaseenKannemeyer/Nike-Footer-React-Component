
# React Footer Component

A responsive **React Footer** component inspired by modern e-commerce layouts (Nike-style), built with **Tailwind CSS** and **Lucide React icons**.  
It features a **mobile accordion layout** and a **desktop grid layout**, ensuring a clean and accessible experience across all screen sizes.

## Features

-   **Responsive Design**
    
    -   Mobile-first accordion layout using collapsible sections.
        
    -   Desktop grid layout with clearly separated footer categories.
        
    -   Fully responsive using Tailwind breakpoints (`md:`).
        
-   **Mobile Accordion Navigation**
    
    -   Smooth open/close animations using CSS transitions.
        
    -   Single open section at a time for better UX.
        
    -   Chevron icon rotation for visual feedback.
        
-   **Desktop Footer Grid**
    
    -   Multi-column layout displaying all footer links at once.
        
    -   Dedicated section for country/location display.
        
    -   Clean spacing and typography for readability.
        
-   **Bottom Utility Bar**
    
    -   Copyright notice.
        
    -   Legal and informational links.
        
    -   Aligned brand logo with responsive positioning.
        
-   **Icons**
    
    -   Uses [Lucide React](https://lucide.dev/) for lightweight, modern icons.
        
    -   Includes globe icon for country selection.
        
-   **Tailwind CSS Styling**
    
    -   Fully styled with Tailwind utility classes.
        
    -   Easy to customize colors, spacing, and typography.
        
    -   No external CSS files required.

## Installation

Install the required dependencies:

`npm install react lucide-react` 

Ensure **Tailwind CSS** is properly set up in your project.


----------

## Customization

-   **Footer Links**  
    Update sections and links inside the `footerLinks` array.
    
-   **Branding**
    
    -   Replace the logo image with your own.
        
    -   Update colors by changing Tailwind utility classes.
        
-   **Country**
    
    -   Modify the displayed country text or extend it into a selector.
        

----------

## Accessibility Notes

-   Uses semantic HTML elements (`footer`, `button`, `ul`, `li`).
    
-   Accordion buttons are keyboard-accessible.
    
-   Clear visual indicators for interactive elements.
